# Clockrr

A clock addon for [Stremio](https://www.stremio.com/) media player.

## Installation

### From URL
```
stremio://localhost:7000/manifest.json
```

### Manual
1. Clone this repo
2. Run `npm install`
3. Run `npm start`
4. Open Stremio → Addons → Community → paste manifest URL

## Development

```bash
# Install dependencies
npm install

# Start addon server
npm start

# Server runs at http://localhost:7000
```
